### 套路交定金

#### 定金个能退，随时退，一个月退。


交订金的时候套路你，一个月退押金。

交首付套路你，我有业绩压力，随时给你退。

有缴款条件好的新客户了，会忽悠你退房，忽悠你写个声请就能退。

然后一房两买了。你定下的房子下家都网签了还不给你办理退款手续。
